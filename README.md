# AI-sales_copilot
# 🚀 AI Sales Copilot for Startups

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://ai-sales-copilot.streamlit.app)

## 📌 Features
- **Lead Scoring** - Automatic lead prioritization
- **Hot Leads Detection** - Identify high-conversion prospects
- **Engagement Tracking** - Monitor email opens, meetings, visits
- **Interactive Dashboard** - Real-time insights

## 🎯 Quick Start

### Option 1: Run Locally
```bash
# Clone repo
git clone https://github.com/shadesvinay01/ai-sales_copilot.git
cd ai-sales_copilot

# Install dependencies
pip install -r requirements.txt

# Run app
streamlit run app.py
